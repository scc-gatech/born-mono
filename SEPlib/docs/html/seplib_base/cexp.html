NAME

cexp - complex exponent

SYNOPSIS

val= complex cexp(a)

INPUT PARAMETER

a - complex
      number to find the exp of

RETURN VALUES

val- complex
      exp(a)

DESCRIPTION

     Cexp returns exp(a) for an arbitrary complex a.  An
     equivalent formula is csmul(ciexp(a.im),exp(a.re)).

SEE ALSO

the csmult manpage,the ciexp manpage

LIBRARY
