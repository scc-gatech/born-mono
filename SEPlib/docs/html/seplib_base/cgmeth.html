NAME

cgmeth - conjugate gradient method

SYNOPSIS

call cgtest( x, yy, rr, fff, niter)

INPUT PARAMETERS

yy - real(:)
rr - real(:)
fff - real(:)

OUTPUT PARAMETERS

x - real(:)

DESCRIPTION

 setup of conjugate gradient descent, minimize  SUM rr(i)**2
             nx
 rr(i)  =   sum fff(i,j) * x(j)  -  yy(i)
            j=1
