NAME

lapfac - factor a 2-D Laplacian

SYNOPSIS

aa= lapfac2(eps,n1,na)

PARAMETERS

eps - real
      small value for approximation
n1 - integer
      length of n1 axis
na - integer
      1/2 number of filter coefs for representation

RETURN VALUE

aa - type(filter)
      Helix filter

DESCRIPTION

Create a one sided, minimum phase filter that approximates a laplaciaan
