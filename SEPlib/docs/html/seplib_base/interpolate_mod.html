NAME

interpolate_mod - Linear interpolation

SYNOPSIS

Initialized=call init_1d_linear_interp(o1out,d1out,n1out,b)

Operator=call interp_1d_linear(input,output,var)

INPUT PARAMETERS

o1out - real
      output axis begining
d1out - real
      output axis sampling
n1out - integer
      number of output points
b -integer
      boundary condition (0-zero outside var boundary, 1 replicate
       last value)
input - real(:)
      Input values
var - real(:)
      Input sample location

OUTPUT PARAMETERS

output - real(:)
      output linear interpolated

DESCRIPTION

Interpolate an irregular sampled function to a regular function.

LIBRARY

sepauxf90
