NAME

steering - 2-D steering filters

SYNOPSIS

call steer_init(hwidth,amp,n1)

call steering2(slope,aa)

PARAMETERS

hwidth - real
      half-width of triangle
amp - real
      amplitude that all coefs sum to
n1 - integer
      Axis 1 length
slope - real
      slope to make filter destroy
aa - type(filter)
      Helix filter

DESCRIPTION

Create a single steering filter

LIBRARY

geef90
