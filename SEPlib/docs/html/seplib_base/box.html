NAME

box - filter to hypercube

SYNOPSIS

call boxn (nd,center,na,aa,cube)

INPUT PARAMETERS

nd - int(:)
      Input data dimension
center - int(:)
      Location of the 1 coeficient in box
na - int(:)
      Dimensions of the filter
aa -
      C<type(filter)> The filter

OUTPUT PARAMETERS

cube - real(:)
      Output filter values

DESCRIPTION

 Convert helix filter to hypercube: cube(na(1),na(2),...)

SEE ALSO

helix, cartesian, helixcart

LIBRARY

geef90
