NAME

createhelix - create a helix filter

SYNOPSIS

aa=createhelix(nd,center,gap,na)

INPUT PARAMETERS

nd - integer(:)
      size of data
center - integer(:)
      location of the 1 of the filter within na box
gap - integer(:)
      distance along each axis before filter coef
na - integer(:)
      size of box around filter

RETURN VALUE

aa - type(filter)
      Helix filter

DESCRIPTION

Create helix filter

SEE ALSO

the createnhelix manpage,the helix manpage,the createmshelix manpage

LIBRARY

geef90
