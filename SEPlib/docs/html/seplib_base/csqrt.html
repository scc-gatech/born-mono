NAME

csqrt - Complex square root

SYNOPSIS

val= csqrt(a)

INPUT PARAMETERS

a - complex
       input number

RETURN VALUES

val - complex
      result of sqrt

DESCRIPTION

    Complex square root
     Csqrt returns sqrt(a) for an arbitrary complex a.  An
     equivalent formula is cspow(a,0.5).

LIBRARY

sepmath
