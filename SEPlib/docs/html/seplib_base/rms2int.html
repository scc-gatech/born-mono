NAME

rms2int - convert to and from rms/interval velocity

SYNOPSIS

call rms2int(inverse,dt,vrms,nt,vint)

INPUT PARAMETERS

inverse - int
      0-int->rms 1-rms->int
dt - real
      sampling in time
vrms - real(nt)
      RMS velocity
nt - int
      Number of time samples
vint - real(nt)
      Interval velocity

DESCRIPTION

A fortran77 program to convert velocity to and from rms velocity

LIBRARY

sepvelanf
