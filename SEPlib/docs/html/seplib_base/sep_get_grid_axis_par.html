NAME

sep_get_grid_axis_par - grab data's n,o,d and label

SYNOPSIS

ierr=sep_get_grid_axis_par(tag_history,i_axis,n,o,d,label)

INPUT PARAMETERS

char* - tag_history
      Tag of History File
int* - i_axis
      Axis number of the parameters to read

OUTPUT PARAMETERS

int* - n
      value of n-i (i = i_axis)
float* - o
      value of o-i (i = i_axis)
float* - d
      value of d-i (i = i_axis)
char* - label
      value of label-i (i = i_axis)

RETURN VALUES

 -1 = if fails for other reasons
 0 = if successful
 +1 = if tag_history is not a Sep History File

DESCRIPTION

Get the values that describe the GRid Axes from the History File.

COMMENTS

        Defaults the others (o=0 d=1 label="label-i axis").
