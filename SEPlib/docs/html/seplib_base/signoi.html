NAME

SYNOPSIS

call signoi_init(nn,ss,niter,nd,eps)

ierr=signoi_lop(adj,add,data,sign)

call signoi_close()

PARAMETERS

nn type(filter)
       Noise filter
ss type(filter)
       Signal filter
niter integer
      Number of itterations
nd integer
      Number of data points
eps real
      Epsilon
adj,add,data,sign
