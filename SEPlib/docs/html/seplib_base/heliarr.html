NAME

heliarr - two helix convolutions

SYNOPSIS

Initialize: call heliarr_init(nx,aa1,aa2)

Operator : ierr= heliarr_lop(adj,add, xx,yy)

PARAMETERS

aa1,aa2 - type(filter)
       Helix filters
nx1 - integer
       Size of model space where aa1 will be applied
adj,add,xx,yy
       standard operator conventions;

DESCRIPTION

An array of helix convolutions. aa1 Will be applied to points (1:nx1) operator two to the remaining data.

SEE ALSO

the helicon manpage

LIBRARY

geef90
