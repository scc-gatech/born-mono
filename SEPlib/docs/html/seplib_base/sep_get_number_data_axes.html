NAME

sep_get_number_data_axes - get number of data axes

SYNOPSIS

ierr= sep_get_number_data_axes (tag_history,n_axis)

INPUT PARAMETER

char*-tag_history
         tag of History File

OUTPUT PARAMETER

int*-n_axis
         Number of axes

RETURN VALUE

 -1= if fails for other reasons
 0= if successful
 +2= if tag_history is an Sep77 History File

DESCRIPTION

        Get number of axes in history file

SEE ALSO

the sep_get_number_header_axes manpage, the sep_get_number_grid_axes manpage

LIBRARY

sep3d
