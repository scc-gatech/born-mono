NAME

weiwem - one depth step of wave equation migration

SYNOPSIS

PARAMETERS

iws - integer
  current frequency slice
izs - integer
  current depth slice
ifk - integer
  current fk operation
ith - integer
  current thread
wfld - complex(:,:,:,:,:)
  wavefield

DESCRIPTION

Wave Equation Migration operator
