NAME

weight - simple weighting operator

SYNOPSIS

call weight_init(w)

ierr=weight_lop(adj,add,model,data)

PARAMETERS

w - real(:)
      Weight to apply
adj,add,model,data -
      Standard operator interfaces

DESCRIPTION

Simple weighting operator. Model and data must be the same size.

LIBRARY

geef90
