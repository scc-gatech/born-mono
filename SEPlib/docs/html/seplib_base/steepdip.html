NAME

steepdip - Find steep dip decon filter

SYNOPSIS

aa=steepn(n,a,d,vel,tgap)

INPUT PARAMETERS

n - integer(:)
      data size
a - integer(:)
d - real(:)
      sampling
vel - real
      velocity
tgap - real
      time gap

RETURN VALUE

aa - type(filter)
      steep dip filter

DESCRIPTION

Create steep dip decon filter

LIBRARY

geef90
