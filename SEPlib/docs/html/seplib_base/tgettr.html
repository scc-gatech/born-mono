NAME

tgettr - read a specified trace from a SEP3d dataset

SYNOPSIS

ierr= tgettr("in", trace,itr)

INPUT PARAMETERS

sep3dname - char*
      tag of sep_3d structure
trace - segy*
      trace to read in
itr - int
      trace number

RETURN VALUES

returns 0 at end of traces
nread =
      number of bytes read

DESCRIPTION

Read in a given trace specified by itr

SEE ALSO

the tgettr manpage, the tputtr manpage

LIBRARY

sepsu
