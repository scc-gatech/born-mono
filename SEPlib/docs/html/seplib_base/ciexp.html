NAME

ciexp - exponent of imaginary number

SYNOPSIS

val=complex ciexp(f)

INPUT PARAMETERS

f - float
      imagionary number

RETURN VALUE

val - complex
      exp(f) of imaginary number

DESCRIPTION

Ciexp returns exp(i*f) = (cos f, sin f). (For pure imaginary number)

SEE ALSO

the cexp manpage

LIBRARY

sepmath
