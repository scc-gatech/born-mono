NAME

sep_put_number_keys - put number of keys

SYNOPSIS

   int sep_put_number_keys(tag_history,n_keys)

INPUT PARAMETER

char*- tag_history
        tag of History File
int* - n_keys
        number of Header Keys

RETURN VALUE

 -1 = if fails for other reasons
 0 = if successful
 +1 = if tag_history is an Sep77 History File
