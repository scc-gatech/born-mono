NAME

sep3d_grab_headers - grab headers

SYNOPSIS

call sep3d_grab_headers(tag,struct,nh,nwind,fwind,jwind)

INPUT PARAMETERS

tag - char*
      tag to read from
struct- sep3d
      struct to read into
nwind - int*
      number of elements
fwind - int*
      first element
jwind - int*
      sampling of elements

OUTPUT PARAMETERS

nh - int
      number of headers read

DESCRIPTION

Read in headers to structure

SEE ALSO

the sep3d_read_data manpage

LIBRARY

supersetf90
