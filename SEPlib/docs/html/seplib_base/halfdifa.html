NAME

halfdifa - half causal derivative

SYNOPSIS

 call halfdifa(adj,add,n,xx,yy)

PARAMETERS

adj - integer
      1=adjoint, 0=forward
add - integer
       1=add to output, 0=erase output
n - integer
      size of data
xx - real(n)
      model
yy - real(n)
      data

DESCRIPTION

 Half order causal derivative.  OK to equiv(xx,yy)

SEE ALSO

the Halfint manpage

LIBRARY

sepfilterf90
