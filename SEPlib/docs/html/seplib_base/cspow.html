NAME

cspow - raise complex number to real power

SYNOPSIS

val= cspow(c,s)

INPUT PARAMETERS

c - complex
       number to raise
s - real
       power to raise it

RETURN VALUES

val - complex =
       result of c**s

DESCRIPTION

     Raise a complex number to a real power
     Cspow returns the principal value of a**s for an arbitrary
     complex a and real exponent s.  An equivalent formula is
     cexp(s*clog(a)).
