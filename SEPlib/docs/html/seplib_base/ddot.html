NAME

ddot - calculate double precission dot product

SYNOPSIS

product=ddot(arr1,arr2)

INPUT PARAMETERS

arr1,arr2 - real(:)
      arrays to calcuate dot products of

RETURN VALUE

product - double precission
      result of dot product

DESCRIPTION

Calculate a dout product of two real vectors.
