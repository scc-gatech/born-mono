NAME

conv - convolve helix filters

SYNOPSIS

ss=conv(aa,bb)

INPUT PARAMETERS

aa,bb - type(filter)
      Helix filters

RETURN VALUE

ss - type(filter)
       Result of helix filter convolution

DESCRIPTION

Convolves two helix filters and returns result

the helix manpage,the compress manpage,the helicon manpage
