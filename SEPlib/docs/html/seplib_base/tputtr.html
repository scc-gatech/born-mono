NAME

tputtr - put given trace next in output stream

SYNOPSIS

ierr= tputtr("in", trace)

INPUT PARAMETERS

sep3dname - char*
      tag of sep_3d structure
trace - segy*
      trace to read in

RETURN VALUES

 0  = if it fails
 1  =  if it succeeds

DESCRIPTION

Read in the next trace

SEE ALSO

the tgettr manpage

LIBRARY

sepsu
