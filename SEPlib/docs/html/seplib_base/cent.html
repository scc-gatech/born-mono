NAME

cent - nth percentile of an array

SYNOPSIS

val=cent(p,x,n)

INPUT PARAMETER

p - float
       percentile <0.,99.999>
x - float*
       data
n - int
       vector raslength

RETURN VALUES

float =
      value at given percentile

DESCRIPTION

Percentile routine based on Canales, SEP-10

LIBRARY

sepaux
