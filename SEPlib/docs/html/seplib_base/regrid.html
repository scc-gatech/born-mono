NAME

regrid - convert a helix filter from one data space to another

SYNOPSIS

call regridn(nold,nnew,aa)

PARAMETERS

nold - integer(:)
      Old data dimensions
nnew - integer(:)
      New data dimensions
aa - type(filter)
      Helix filter

DESCRIPTION

Converts a helix filter from one ata size to another

SEE ALSO

the cartesian manpage,the helix manpage

LIBRARY

geef90
