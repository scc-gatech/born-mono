NAME

sep3d_local_buffer_size - Get the size of the data associated with a given window

SYNOPSIS

ndim= sep3d_local_buffer_size(struct,n,nwind,fwind,jwind)

INPUT PARAMETERS

struct - sep3d
      structure
n - int (2)
      the output window size
nwind - int*
      [nwind in memory, optional] the number of samples in a window
fwind - int*
      [fwind in memory, optional] the first sample in a window
jwind - int*
      [jwind in memory, optional] the sampling of the window

DESCRIPTION

Get the size associated with a given window (defaults to the window described by the last read_headers call or the entire dataset) Useful for sectioned tags.
