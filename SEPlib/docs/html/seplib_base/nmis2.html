NAME

nmis2 - Fill in missing data using non-stationary filter

SYNOPSIS

call mis1(niter,xx,aa,known,doprec)

PARAMETERS

niter - integer
       number of iterations
xx -real
       fitting variable
aa -type(nfilter)
       filter to apply
known -logical(:)
       Known data
doprec -logical
       Whether or not to run preconditioning

DESCRIPTION

fill in missing data by minimizing power out of a given non-stationary filter by helix magic works in any number of dimensions

SEE ALSO

mis2,msmis,mask1,the helicon manpage,the solver manpage,the nhelix manpage
