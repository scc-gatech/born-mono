NAME

refine2 - refine mesh

SYNOPSIS

call refine2_init(co1,cd1,co2,cd2,m1,m2,fo1,fd1,fo2,fd2,n1,n2)

ierr= refine2_lop(adj,model,data)

call refine2_close()

PARAMETERS

co1,co2 - real
      Coarse grain origin
cd1,cd2 - real
      Coarse grain sampling
ro1,ro2 - real
      Refined grain origin
rd1,rd2 - real
      Refined grain sampling
m1,m2 - integer
      Size of coarse grain input
n1,n2 - integer
      Size of fine grain output
adj,add,model,data -
      Standard operator interface

DESCRIPTION

 Refine a mesh by linear interpolation
