NAME

sep3d_grab_key_vals - grab header values from C structure

SYNOPSIS

call sep3d_grab_key_vals(struct, locate, values) must have already made a call to sep3d_grab_headers('in',input,nh)

INPUT PARAMETERS

struct - sep3d
      structure to grab from
locate - (char*/int)
      locate through keyname or keyindex

OUTPUT PARAMETERS

values - (float*/int*)
              array of header values

DESCRIPTION

Grab header values

SEE ALSO

the sep3d_set_key_vals manpage

LIBRARY

supersetf90
