NAME

autocorr - compute a filter auto-correlation

SYNOPSIS

filter=autocorr1(aa,s0,a0)

INPUT PARAMETERS

aa - type(filter)
      filter to compute auto-correlation of
a0 - real
      zero lag value of input
s0 - real
      zero lag of output filter

RETURN VALUE

filter -
      autocorrelation of helix filter

DESCRIPTION

Compute the autocorrealtion of filter. Useful for input into wilson.

SEE ALSO

the helix manpage,the compress manpage,the wilson manpage

LIBRARY

geef90
