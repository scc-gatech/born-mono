NAME

cartesian - convert to and from cartesian

SYNOPSIS

call line2cart(nn,i,ii)

call cart2line(nn,ii,i)

PARAMETERS

nn - int(:)
      cartesian axes
ii - int(:)
      cartesian coordinates
i - int
      linear index

DESCRIPTION

index transform (vector to matrix) and its inverse

LIBRARY

geef90
