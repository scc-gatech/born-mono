NAME

sep3d_set_sep3d - synchronize C structure with f90 structure

SYNOPSIS

<call sep3d_set_sep3d(sep3df)>

INPUT PARAMETERS

sep3df - sep3d
      fortran sep3d structure to copy from

DESCRIPTION

 Copies sep3d F90 structure to its C equivilant

SEE ALSO

the sep3d_grab_sep3d manpage

LIBRARY

supersetf90
