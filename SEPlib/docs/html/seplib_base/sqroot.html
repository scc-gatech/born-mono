NAME

sqroot - square root

SYNOPSIS

val=sqrt(r,t0,epsilon)

INPUT PARAMETERS

r - real
      number squared
t0 - real
      initial value
epsilon - real
      desired precision

RETURN VALUE

val - real
      square root

COMMENTS

Finds the square root of r, starting from t0.

A simple Newton algorithm. The hope is that with a proper choice of t0, it makes less iterations than the system sqrt.

DESCRIPTION

``Fast'' square root

LIBRARY

sepmath
