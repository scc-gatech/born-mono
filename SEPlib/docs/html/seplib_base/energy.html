NAME

energy - calculate energy

SYNOPSIS

ierr= energy (intag,outtag,data,data2,n1,n2,n3,lwind,j1,n1new,norm,ave)

INPUT PARAMETERS

intag- char*
      Input tag
outtag - char*
      Output tag
data - real(n1,n2)
      Input offset-time gather
data2 - real(n1new,n2)
      Output velocity-time gather
n1- integer
      Input length1
n2 - integer
      Length axis2
n3 - integer
      Length axis3
lwind - integer
      Length of energy window
j1- integer
      Spacing betweew centers of energy windows
n1new - integer
      Output fast axis length
norm - integer
      Flag for normalizing energy (0,1)
ave - integer
      Flag for normalizing window length

DESCRIPTION

  Subroutine to calculate energy in running windows along fast data axis
