NAME

csmult - complex multiplication

SYNOPSIS

val= csmult(c, s)

INPUT PARAMETERS

c - complex
        complex input
s - float
        float values to multiply by

RETURN VALUES

val- complex
       result of multiplication

DESCRIPTION

    Multiply complex number by real scalar
     Csmul returns c*s.

the cmult manpage
