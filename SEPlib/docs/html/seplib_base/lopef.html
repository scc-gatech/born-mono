NAME

lopef - estimate pef in patches

SYNOPSIS

call find_lopef(wall,aa,npatch,nwall,nwind,mask)

PARAMETERS

wall - real(:)
aa - type(filter)
       helix filter
npatch -integer(:)
       size of patch
nwall -integer(:)
nwind -integer(:)
mask -real(:) 
      Mask of known and unknown data

DESCRIPTION

Estiamte a PEF in patches
