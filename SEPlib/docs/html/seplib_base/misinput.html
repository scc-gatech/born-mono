NAME

misinput - find a mask of missing filter inputs

SYNOPSIS

call find_mask(known,aa)

PARAMETERS

known - logical(:)
       Known data locations
aa - type(filter)
       Helix filter

DESCRIPTION

Given known locations of data, mark on filter where convolution result is valid.

SEE ALSO

the lopef manpage,the helicon manpage

LIBRARY

geef90
