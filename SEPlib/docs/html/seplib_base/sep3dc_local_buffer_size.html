NAME

sep3dc_local_buffer_size - Local buffer size given a window

SYNOPSIS

logic=sep3dc_local_buffer_size(sep3d *sep3din, int *n,int *nwind, int *fwind, int*jwind)

INPUT PARAMETERS

sep3din - sep3d
     structure
n - int[2]
    Output buffer size
nwind - int*
    Number of elements along each axis
fwind - int*
    First  sample along each axis
jwind - int*
    Sampling of elements along each axis

DESCRIPTION

Distribute the headers to a section tag (space2) from a non-sectioned tag (space1)
