NAME

Hwt3d - 2-D ray tracing

SYNOPSIS

< velocity.H Hwt2d par= > rays.H

DESCRIPTION

Does Huygens wavefront tracing. Produces a ray database.

INPUT PARAMETERS

SHOT LOCATION

xsou - float
zsou - float

RAY PARAMETERS

oT - float
      [0.]
nT - int
      [100] number of traveltime steps
dT - float
      [0.01] traveltime sampling
oG - float
      [0.]
nG - int
      [90] number of shooting directions
dG - float
      [1.0] shooting-angle sampling
