NAME

In3d - info in seplib datasets

SYNOPSIS

In3d header.H [header2.H .....] [pars]

Input parameters

keys- character
       [y] print out header keys
maxkeys - integer
        [number of keys] maximum number of keys to print
grid character
        [y]  print grid axis if they exist
check character
        [y]  check for zeros in file
dff- character
       [n] print out dff info
maxkeys - integer
        [nkeys] Number of keys to print

DESCRIPTION

Provide useful information about SEP3d datasets

SEE ALSO

In, the Attr manpage, the Attr3dhead manpage
