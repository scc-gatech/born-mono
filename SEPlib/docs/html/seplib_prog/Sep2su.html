NAME

Sep2su - convert from sep3d to su

SYNOPSIS

Sep3d2su < in.H >out.su pars

INPUT PARAMETERS

verb - int [1000000]
      How often to print number of traces converted

DESCRIPTION

Converts a SEP3d dataset to SU
