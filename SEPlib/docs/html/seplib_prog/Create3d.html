NAME

Create3d - Create a SEPlib3d dataset

SYNOPSIS

 Create3d <in.H >out.H  [headers= ] [keyname1= keyname2 ....)

INPUT PARAMETERS

headers - sepfile
       file containing headers
keyname_ - int
       defaults to key number (nkeys must equal n1 
       of headers)
verb - int
       [0]  whether or not to be verbose

DESCRIPTION

Program that creates a SEPLIB3d file from either two S77 files or a single s77 file

WARNING! The input file headers must NOT be out.H@@

CATEGORY
