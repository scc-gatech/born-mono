NAME

util/cube - Programs that take advantage of the SEP hypercube

SYNOPSIS

A SEPlib library

DESCRIPTION

Programs that take advantage of the SEP hypercube

PROGRAMS

the Scat manpage - Programs that take advantage of the SEP hypercube

the Merge manpage - Concatenate seplib datasets, 5 names for the same exact code.

the Interleave manpage - Interleave 2 seplib datasets

the Patch manpage - Create patches (or combine patches)

the Pad manpage - Pad a dataset with traces of zeros or nearest value

the Window manpage - Window a dataset by cutting or decimation

the Reverse manpage - reverse (switch direction of) axes

the Window3dM manpage - Window a SEPlib dataset (3-D or 2-D) by cutting and/or decimating samples.

the Reshape manpage - Reshape a SEPlib dataset

the Transp manpage - transpose a SEPlib dataset in plane of choice
